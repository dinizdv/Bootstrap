-> What are breakpoints?
> They are responsive building blocks with predefined widths for different viewport sizes.

-> All Bootstrap breakpoints:
> extra-small == < 576px
> small == sm == >= 576px
> medium == md == >= 768px
> large == lg == >= 992px
> extra-large == xl >= 1200px
> extra-extra-large == xxl >= 1400px
